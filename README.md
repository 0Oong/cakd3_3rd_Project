### 프로젝트 기대 가치
1) 재활용률 상승  : 분리배출 기준을 명확히 제시하여 수거율 및 재활용률을 높인다.
2) 사용자 인식개선 : 분리배출 과정에서 사용자의 인식개선이 이루어진다.
3) 기업의 생산방식 변경 : 사용자의 인식개선에 따라 기업의 생산방식도 재활용이 편리한 방향으로 바뀌는 선순환이 이루어진다.
